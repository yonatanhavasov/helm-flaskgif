Helm Flask GIF Repository

Welcome to the Helm Flask GIF repository! Use the commands below to add this Helm chart repository and install the chart in your Kubernetes cluster:

Steps to Use the Chart

  1. Add the repository: 
    helm repo add <reponameofyourchoice> https://yonatanhavasov.github.io/helm-flaskgif
  2. Update the Helm repositories: 
    helm repo update
  3. Install and Run the chart: 
    helm install <releasenamechoice> <reponame>/flask-app - flask-app is the chart name

Chart Files

  • index.yaml - Helm chart repository index file.
  • helm-flask-0.1.0.tgz - Packaged Helm chart.

For more information about Helm, visit the Helm official documentation.
